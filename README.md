# Validation Form

This project is a React-based registration form application that includes form validation and user-friendly features such as placeholder text in input fields.

## Features

- User registration form with fields for personal information, account details, contact information, and identity verification.
- Form validation for required fields, email format, phone number, PAN number, and Aadhar number.
- Responsive and styled using Tailwind CSS.
- Local storage saving of registration data upon successful submission.

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Anixh06/Validation_form.git
cd Validation_form
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to:

```
http://localhost:5174/
```

### Usage

- Fill in the registration form with the required information.
- Placeholder text in each input field guides you on what to enter.
- Submit the form to save your data and proceed to the success page.

## Project Structure

- `src/components/` - Reusable form input components.
- `src/pages/` - Application pages including the Registration form.
- `src/utils/` - Utility functions for validation.
- `src/data/` - Static data such as country and city lists.
